# 基于JavaFX实现的公园游船租船管理系统

## 项目运行
打开项目后运行src/main路径下的HelloApplication.java文件启动

## 项目打开方法1
直接 Download Zip 下载项目源代码后解压，在IDEA中打开运行

## 项目打开方法2
在IDEA中新建一个JavaFX项目，注意**工件**名称处需修改为boat,否则需要在代码中逐个修改为自己命名的包名；JDK下载或选择JDK8级以后版本，创建项目
